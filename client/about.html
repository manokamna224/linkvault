ℹ️ About this project

What is LinkVault?

A free, open, community-powered repository of hand-picked links for developers, designers, and curious builders.

🎯

Our Mission

Developers and designers waste hours searching for quality resources scattered across Reddit threads, Twitter posts, and disorganised bookmark folders. LinkVault solves this by creating one trusted, community-validated home for the best links in every niche — curated by real people, ranked by votes.

⚙️

How It Works

1

Anyone submits a link

Register for free and submit any resource you find valuable. It enters a moderation queue.

2

Moderators review it

Our admin team reviews each submission for quality and relevance before it goes live.

3

Community votes

Approved links can be upvoted or downvoted. Scores update in real-time via WebSockets.

4

Best rises to the top

Sort by Top Rated, Newest, or Controversial. Filter by 14 categories and 40+ tags.

✨

Key Features

🔌

141+ Curated Links

Hand-picked resources across 14 categories including Free APIs, Dev Tools, AI Prompts, and more.

⚡

Real-Time Voting

Upvote and downvote links instantly. Scores update live for all users via WebSockets.

🔍

Powerful Search

Filter by keyword, category, tag, sort order, and date — all simultaneously.

🛡️

Moderated Quality

Every link is reviewed before going live. No spam, no low-quality content.

🌙

Full dark mode support. Your preference is saved across sessions.
